# mooresolutions.io

## About this project

I have decided to redesign my personal website. The main goal is to make a website that is visually striking, and technological impressive.

## About me
My name is Daniel Moore. I started web development in 2014. I have a passion for building websites and applications that 
are both functional and visually appealing. I specialize in back-end development using multiple technologies, but I am 
also proficient in front-end development. I'm currently focusing on native mobile development using Swift and Kotlin. 

## Features
I would like a website that can be built into static HTML, CSS, and JavaScript. I would like a directory where I can 
place Markdown files that will be converted into HTML and displayed as blog posts on the site. It should use Vue3 for frontend components. 
I would also like a similar directory structure for projects.

## Design
I would like a universal dark theme that attempts to mimic a terminal / TUI. 100% of the website should be responsive and be able to accommodate touch devices.
The purpose of this website is to showcase projects and to share my thoughts on various topics related to technology and programming.
I don't care about clients / conversion. I only want to showcase my work and my thoughts.

I would like the color scheme to mimic the oh-my-zsh halflife theme.

## Deployment
I would like to deploy this website on GitHub Pages. I would like to use GitHub Actions for continuous deployment. I would like to use a custom domain name that I have purchased through Namecheap. I would like to use Cloudflare for DNS management.

## Considerations
The website should be as lightweight as possible.
It should be technically impressive.
It should be visually appealing.
It should be mobile-friendly.
Any icons should use SVGs and be optimized for performance and generated using PIL (Python Imaging Library).
It should be responsive.
It should be SEO-friendly.
It should be accessible.
It should be fast.
